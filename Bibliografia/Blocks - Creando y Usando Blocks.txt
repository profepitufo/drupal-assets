BLOCKS

Un contenedor de informacion que se despliega en una region de la pagina o paginas.

VISUALIZAR LAS REGIONES DE UN TEMA.
Structure / Block Layout / Demonstrate block regions (del tema que se tenga).

La manera mas comun es mostrarlos en los sidebar.

Existen 4 tecnicas de crear blocks:
	1. Blocks incluidos en Drupal Core
	2. Blocks creados en Modulos Externos
	3. Blocks creados por Vistas y Menus
	4. Blocks creados Custom

CREAR UN BLOCK
Structure / Block Layout / Custom Block Library
Add custom block
Block description: Colocar una Descripcion para el Block.
Body: Escribir contenido.
SAVE.

AGREGANDO BLOCK A UNA REGION
Structure / Block layout
Sidebar second, Place Block
*localizar un block, configurarlo de manera adecuada a sus necesidades.
SAVE BLOCK.
SAVE BLOCKS.

CONTROL BLOCK VISIBILITY
NOTA: acceder al block utilizando el icono del lapiz, en caso de estar en modo edicion del lugar.
